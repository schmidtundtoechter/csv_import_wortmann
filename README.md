## Csv Import Wortmann


The app generates sales invoices in ERPNext from the CSV of Wortmann to bill the final customers.

Features:
* Discounts can be entered per customer.
* Invoices without costs for the customer can be suppressed.
* Quick links and a history of the imports.
* Import of the CSVs to a special folder in ERPNexts Data DocType.
* An account for VAT can be specified.
* A due date by which the payment must be received is specified.
* An import report checks whether all lines have been processed correctly, provides a statistical evaluation and suggests solutions problems occure.
